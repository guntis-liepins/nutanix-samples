Create project directory
```
ctrlz@vulcan:~/Projects/Python$ mkdir nutanixapi-test
```
Change to this directory and then crate virtual environment in venv folder
(naming it venv can help Visual Studio code to work with virtualenv)
```
ctrlz@vulcan:~/Projects/Python$ cd nutanixapi-test/
ctrlz@vulcan:~/Projects/Python/nutanixapi-test$ python3 -m venv venv
```
Activate python project:
```
ctrlz@vulcan:~/Projects/Python/nutanixapi-test$ source venv/bin/activate
```
Upgrade pip 
pip install --upgrade pip

In project folder make file requirements.txt which contain project dependencies and 
add nutanixapi  repository url as dependency there among other packages.

git+git://github.com/guntis-liepins/nutanixapi.git

Then install dependencies
First install wheel package. This is unsolved bug ,wheel installed after nutanixapi so nutanixapi install will fail.
Need to be done only once when new virtualenv is created.
Then install requirements
```
pip install wheel
pip install -r requirements.txt
```
